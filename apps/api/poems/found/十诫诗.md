# 十诫诗

_author: 仓央嘉措
_date: 1700


第一最好不相见，如此便可不相恋。
第二最好不相知，如此便可不相思。
第三最好不相伴，如此便可不相欠。
第四最好不相惜，如此便可不相忆。
第五最好不相爱，如此便可不相弃。
第六最好不相对，如此便可不相会。
第七最好不相误，如此便可不相负。
第八最好不相许，如此便可不相续。
第九最好不相依，如此便可不相偎。
第十最好不相遇，如此便可不相聚。
但曾相见便相知，相见何如不见时。
安得与君相诀绝，免教生死作相思。


First The best would be never to meet,
so we would never fall in love.

Second The best would be never to know,
so we would never long or yearn.

Third The best would be never to stay,
so we would owe each other nothing.

Fourth The best would be never to cherish,
so no memories would cling or ache.

Fifth The best would be never to love,
so we would never have to part.

Sixth The best would be never to face,
so we would never have to meet again.

Seventh The best would be never to mistake,
so neither would have to bear regret.

Eighth The best would be never to vow,
so no promise would need breaking.

Ninth The best would be never to lean,
so we would never press heart to heart.

Tenth The best would be never to meet at all,
so we would never have to part at all.

Yet—
since once we met, we came to know;
and seeing each other—how could we not?

If only I could bid you farewell forever,
and save this life and death from longing still.


————

First, the best would be never to meet,
so love would never bloom, nor bleed.

Second, the best would be never to know,
so yearning’s fire would never grow.

Third, the best would be never to stay,
so hearts would owe no debt one day.

Fourth, the best would be never to care,
so memory leaves no pain to bear.

Fifth, the best would be never to love,
so parting tears would not thereof.

Sixth, the best would be never to face,
so fate would never weave our trace.

Seventh, the best would be never to stray,
so none would wrong the other’s way.

Eighth, the best would be never to vow,
so broken faith need not be now.

Ninth, the best would be never to lean,
so closeness need not come between.

Tenth, the best would be never to meet at all,
so never to part, nor rise, nor fall.

Yet once we met, we came to know;
and seeing each, how not to glow?
If I could bid farewell, and keep—
this life and death from endless weep.
